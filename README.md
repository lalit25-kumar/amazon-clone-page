# 🛒 Amazon Clone (HTML & CSS)

This is a simple **Amazon Clone** project built using **HTML and CSS**.
The goal of this project is to practice frontend development and create a visually similar layout to the Amazon homepage.

---

## 🚀 Features

* 🏠 Amazon-like homepage layout
* 📦 Product sections with images
* 🔍 Search bar UI
* 🧭 Navigation bar
* 📱 Responsive design (basic)
* 🎨 Clean and modern UI using CSS

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 📂 Project Structure

```
amazon-clone/
│── index.html
│── style.css
│── images/
```

---

## ▶️ How to Run the Project

1. Download or clone this repository:

   ```
   git clone https://github.com/lalit25-kumar/amazon-clone-page.git
   ```
2. Open the project folder
3. Double click on `index.html` or open it in your browser
